## Installation

Clone the repository:

```
$ git clone https://github.com/0xd3lbow/aggr.template.git
```
## Importing Script

* Clone the Repository or Download the Zip File
* Open https://charts.aggr.trade/
* Select Settings
* Select +New
* Upload the BNB.js Document
